# STOP DYING THERE!!!

<img src="logo.png" width="150" alt="the mod's logo" />

This mod will play lines from Cold's iconic [video](https://www.youtube.com/watch?v=kNDpLgaQQqk) whenever you die in GD. 
The line chosen to play will scale in intensity depending on how far you are in the level (so early deaths have "calmer"
reactions, new bests have all of the, well, least calm reactions)

Specifics and toggles can be found in the mod settings. Have fun with this lol
